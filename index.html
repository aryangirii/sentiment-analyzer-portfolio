Sentiment Analyzer on AWS

Real-time user feedback analysis using AWS services

📌 Project Overview

This project analyzes customer feedback in real-time using AWS Lambda, Amazon Comprehend, and stores results in DynamoDB. A custom React frontend displays feedback with sentiment visualizations and supports dark mode, email alerts, and data insights.

🛠 Technologies Used

  • React.js (Frontend)
  • API Gateway
  • AWS Lambda (Node.js/Python)
  • Amazon Comprehend (NLP)
  • DynamoDB (Storage)
  • Amazon SES (Email Alerts)
