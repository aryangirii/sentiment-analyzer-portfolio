Sentiment Analyzer

Real-Time Customer Feedback Intelligence

Project Overview

Sentiment Analyzer is a production-ready application designed to help organizations understand and act on customer feedback in real time. Using a fully serverless architecture built on AWS, the platform analyzes sentiment using natural language processing and provides live visual feedback through a React-based dashboard. The solution is scalable, cost-effective, and highly actionable—designed for businesses that value customer experience.

Technologies & Services

React.js (Frontend) · AWS Lambda (Node.js/Python) · Amazon Comprehend (NLP) · Amazon API Gateway · DynamoDB · Amazon SES · AWS S3 + CloudFront

Architecture Overview

Feedback is submitted via a web form hosted on AWS S3. The data is processed through Amazon API Gateway and AWS Lambda, where sentiment analysis is performed using Amazon Comprehend. The results are stored in DynamoDB. If a negative sentiment is detected, an alert email is automatically triggered through Amazon SES. An interactive React dashboard fetches and visualizes the data for real-time insights.

Key Features

  • Real-time sentiment classification using Amazon Comprehend
  • Automated email alerts for negative feedback using Amazon SES

  • Interactive React dashboard with sentiment charts and summaries
  • Secure, persistent storage in DynamoDB
  • Modern UI with light/dark mode support

Impact

This solution addresses a real-world business need: transforming raw customer feedback into meaningful insights. With real-time alerting and sentiment trends, it enables product and support teams to proactively address concerns and improve user satisfaction.
