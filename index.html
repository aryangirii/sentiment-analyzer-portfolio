Sentiment Analyzer – Real-Time Customer Feedback Intelligence

This project is a fully deployed, production-ready application that allows organizations to collect, analyze, and visualize customer feedback in real time. Built on a serverless AWS architecture, it uses Amazon Comprehend to extract sentiment from user input and presents the results on an interactive React dashboard. It is designed for high scalability, ease of use, and actionable insights, helping businesses improve customer experience through real-time feedback monitoring.

Technologies & Services

React.js (Frontend) · AWS Lambda (Node.js/Python) · Amazon Comprehend (NLP) · Amazon API Gateway · DynamoDB · Amazon SES · AWS S3 + CloudFront

Architecture Overview

Users submit feedback through a form hosted on AWS S3. This data is routed via API Gateway to a Lambda function, which performs sentiment analysis using Amazon Comprehend and stores the results in DynamoDB. If negative sentiment is detected, Amazon SES sends an alert email. A React.js dashboard pulls this data to display live sentiment trends and breakdowns.
